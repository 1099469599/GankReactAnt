A web repository for study.

Continuous updating...

### Request for

- zepto

### Setup

```
git clone https://github.com/linkaipeng/GankReactAnt

cd GankReactAnt

$ npm install
$ npm install antd
$ npm install css-loader
$ npm install style-loader
$ webpack
$ gulp

visit http://127.0.0.1:8000
```
